Frequently asked questions

How does sharing work without an account or server?

Your entire quiz (title, questions, options, and correct answers) is encoded directly into the link itself. When someone opens it, their browser decodes the quiz from the link — nothing is stored on a server.

Can people cheat by viewing the page source?

The correct answers are technically embedded in the link so the quiz can be scored instantly. This tool is best for casual practice and fun quizzes, not high-stakes exams.

How many questions can I add?

There's no hard limit, but very long quizzes create longer links. For sharing convenience, we recommend keeping quizzes under 20 questions.

Can I edit a quiz after generating the link?

Yes. Go back to the quiz maker, adjust your questions, and generate a new link — the old link will still work with the original questions.

Does it work on mobile?

Yes. Both building and taking a quiz work fully on mobile browsers, with no app install required.
