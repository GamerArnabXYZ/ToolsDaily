Frequently asked questions

      

        What does the JSON formatter do?
        
It takes raw JSON and reformats it with proper indentation (beautify), or condenses it (minify), while validating the syntax.

      

      

        How do I know if my JSON is valid?
        
The status line reports whether the input is valid JSON and shows an error message if something is wrong.

      

      

        Is my JSON uploaded?
        
No. Formatting happens locally in your browser, so your data stays private.

      

      

        Can I load a sample to test?
        
Yes. The 'Load sample' button fills the editor with example JSON so you can try it immediately.

      

      

        Does it tell me exactly where my JSON is broken?
        
Yes. Invalid JSON is flagged with a clear error message so you can find and fix the exact issue instead of guessing.
