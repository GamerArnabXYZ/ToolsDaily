Frequently asked questions

Is my image uploaded to a server?

No. Compression happens entirely on your device using your browser's built-in image tools — the file never leaves your phone or computer.

What image formats are supported?

JPG, PNG, and WebP images can be uploaded. The compressed output is saved as a JPG for the smallest reliable file size.

What quality setting should I use?

75–85% usually gives a good balance of quality and file size. Go lower (40–60%) for thumbnails or web previews where sharpness matters less.

Will compressing reduce the image dimensions?

No. Only the file size (via JPG quality) is reduced — width and height stay exactly the same as the original.

Can I compress a PNG with transparency?

The output format is JPG, which doesn't support transparency, so transparent areas will be filled with white in the compressed result.
